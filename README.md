# SHS Enrollment System

A comprehensive Laravel-based enrollment management system for Senior High School (SHS) students. This system streamlines student registration, payment processing, scheduling, and academic management.

## Features

- **Student Registration** - Manage student enrollment and registration records
- **Payment Processing** - Track and approve student payments
- **Class Scheduling** - Organize course schedules and classroom assignments
- **Strand Management** - Organize students by academic tracks (e.g., STEM, ABM)
- **Section Management** - Create and manage class sections
- **Subject Management** - Maintain course catalog and subject offerings
- **Teacher Management** - Assign teachers to subjects and schedules
- **Room Management** - Manage classroom and facility assignments
- **User Roles & Permissions** - Admin, Cashier, Operator, Teacher, and Employee roles
- **Email Notifications** - Automated payment approval notifications

## Tech Stack

- **Backend**: Laravel 11
- **Frontend**: Tailwind CSS, Vite, JavaScript
- **Database**: SQLite (default) or MySQL/MariaDB
- **PHP**: 8.2 or higher
- **Package Manager**: Composer, npm

## Requirements

- PHP 8.2+
- Composer
- Node.js & npm
- Git
- SQLite or MySQL database
